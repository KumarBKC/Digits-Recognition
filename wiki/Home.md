# Optical Character Recognition (OCR) for Digit Recognition

## Project Overview  
The **Optical Character Recognition (OCR)** project aims to recognize both written and printed digits with high accuracy. This technology is pivotal in automating the digitization of handwritten forms, invoices, and various documents that include numerical data.

## Features  
- **High Accuracy**: Utilizes advanced machine learning techniques to achieve accurate recognition results.
- **Supports Multiple Formats**: Capable of recognizing digits from images, PDFs, and scans.
- **User-Friendly Interface**: Simplifies the process of digit recognition for end-users.
- **Customizable**: Users can train the model further based on specific needs.

## Technology Stack  
- **Programming Language**: Python  
- **Frameworks**: TensorFlow, Keras  
- **Image Processing**: OpenCV  
- **Database**: SQLite (for storing recognition results and logs)

## Installation  
To get started with the OCR project, clone the repository and install the required dependencies using:
```
git clone https://github.com/KumarBKC/Digits-Recognition.git
cd Digits-Recognition
pip install -r requirements.txt
```

## Usage  
After setting up the environment, you can start using the digit recognition features with the provided scripts in the `src` folder. Ensure to place the images in the designated `input` folder, and run:
```
python main.py
```
